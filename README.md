# RPG-
모바일 게임 바람의나라:연 자동사냥에 대한 챗봇
기본적인 CNN 이미지 구분 모델을 이용하여 자동사냥중인지 죽었는지 판단한다.
장비 내구도가 다 닳았는지는 따로 구분하도록 한다.
텔레그램 챗봇을 이용하며 따로 이용시 Bothi Class의 ID와 TOKEN을 수정하여 사용한다.
그 외에 경로와 학습시킬 이미지들을 형식에 맞게 고쳐준다.


실행방법
